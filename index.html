How to make a fake text conversation video

  1. Pick a chat app (WhatsApp, iMessage, Instagram…) and a phone model.
  2. Set the participants' names and photos.
  3. Type the messages for each side — reorder or edit any time.
  4. Watch the live animated preview with typing indicators.
  5. Choose an export resolution (720p, 1080p, 2K or 4K) and press Export Video.
